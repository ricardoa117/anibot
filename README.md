# Anibot

Anibot is an educational Telegram bot focused on promoting sexual health and preventing sexually transmitted infections (STIs). This bot aims to provide reliable information in an accessible manner.

## Features
- **Educational Content**: Provides information on various aspects of sexual health and STI prevention.
- **Interactive Q&A**: Users can ask questions and receive well-researched answers.
- **Resource Links**: Gain access to trusted resources and further reading materials.
- **User-Friendly Interface**: Designed to be easy to use on Telegram, making it accessible for all users.

## Installation Instructions
### Requirements
- A Telegram account.
- Python 3.7 or higher.
- Pip for package management.

### Steps to Install
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your_username/anibot.git
   ```  
2. **Navigate into the directory**:
   ```bash
   cd anibot
   ```  
3. **Install the required packages**:
   ```bash
   pip install -r requirements.txt
   ```  
4. **Set up your Telegram Bot**:
   - Go to [BotFather](https://t.me/botfather) on Telegram.
   - Create a new bot and obtain your API Token.

5. **Configure the Bot**:
   - Create a `.env` file in the root directory with the following:
     ```
     TELEGRAM_API_TOKEN=your_token_here
     ```
6. **Run the bot**:
   ```bash
   python bot.py
   ```

## Usage Guide
- To start using the bot, open Telegram and search for your bot by its username.
- Type `/start` to initiate a conversation with the bot.
- You can ask questions about sexual health and STIs by simply typing your questions.
